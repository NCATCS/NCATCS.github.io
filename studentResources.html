Chris' Reading List

Sometimes, computer science textbooks are too expensive, or too hard to read! Here are some that aren't:

  • Grokking Algorithms
    This is a GREAT book about basic algorithms that explains some complex topics in a very approachable way. 10/10
  • Open Data Structures
    A higher level but absolutely free resource for learning basic data structures. There are Java, C++, and pseudocode/python versions.
  • Pro Git
    A free book that explains everything you've ever wanted to know about git, and about 1,500 things you'd have never thought to ask. Also, FREE.
